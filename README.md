MPD Helper
==========

Simple app to sequence jobs on procesors via sequencing alghoritms, done with C++ and Qt.

###What's done?
Simplest of all alghoritms - Johnson's alghoritm for sequencing.
Simple Qt UI with plotting the resoult.
